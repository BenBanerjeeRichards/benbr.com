My Spotify history

BenBanerjeeRichards/Spotify-history-downloader

According to Spotify wrapped for 2018, I spend 11% of my life listening to music (that's 60,000 minutes last year). Naturally, I wanted to analyse my listening history but Spotify only allows you to download the last 50 songs that you listened to. What started as a small hourly cron script quickly grew more features including the ability to import listening using Spotify's GDPR data export and export data to Github for version control.

You can see my latest creation: my year in music in a poster, filling a gap on a blank wall in my room. I spend a week designing an algorithm through trial and error to lay out the poster in a aesthetically pleasing way, with my favourite albums spaced out properly.

Another creation from my Spotify data is a massive genre cloud showing my favourite genres. It's great to know that I absolutly love genres such as "anthem emo" and "australian garage punk".

R&D is continuing into producing more insightful analysis for this site.

BenLang - It's turing complete!

BenBanerjeeRichards/BenLang

Last year I realized the world needed something - a new programming language. And so BenLang was born. A somewhat featured programming language for which I built a compiler for, using ANTLR 4 and python. The compiler produces MIPS assembly as it's output.

The language includes basic features such as loops, conditionals, variables, type checking and I/O.
